## How to Use

### Run an Extension Exercise
```bash
cd flutter-live-coding-challenges/extensions/001-movie-length
dart run
```

### Run a Widget Exercise
```bash
cd flutter-live-coding-challenges/widgets
flutter run
```

### View Solutions
```bash
cd flutter-live-coding-challenges/extensions/solutions
cat 001-movie-length.dart
```

### Run Tests (when available)
```bash
flutter test
```

## Tips for Success

1. **Start Easy**: Begin with Level 1 exercises to build confidence
2. **Time Yourself**: Simulate interview pressure
3. **Write Tests**: Verify your solutions work
4. **Review Solutions**: Compare with provided solutions
5. **Understand Why**: Don't just memorize - understand the concepts
6. **Practice Daily**: Consistency beats cramming
7. **Mock Interviews**: Practice explaining your thought process
