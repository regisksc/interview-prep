# Flutter Live Coding Challenges

> 500+ exercises for Flutter/Dart interview preparation

## Structure

Each exercise contains:
- **Description**: Clear problem statement
- **Time limit**: Suggested completion time (5/10/15/30 min)
- **Difficulty**: Level 1-5 (Very Easy to Very Hard)
- **Examples**: Input/output test cases
- **Starter code**: Beginning implementation to complete
- **Solution**: Reference solution in `solutions/` subfolder

## Difficulty Levels

| Level | Description | Time | Topics |
|-------|-------------|------|--------|
| 1 - Very Easy | Basic syntax, simple operations | 5 min | String manipulation, basic collections |
| 2 - Easy | Single concept application | 5-10 min | Extensions, basic widgets |
| 3 - Intermediate | Multiple concepts combined | 10-15 min | State management, async patterns |
| 4 - Hard | Complex logic, optimization | 15-30 min | Streams, isolates, custom paint |
| 5 - Very Hard | Architecture, edge cases | 30 min+ | Performance, platform channels |

## Usage

```bash
# Run a specific challenge
cd flutter-live-coding-challenges/extensions/01-string-extensions
dart run

# Run tests for a challenge
flutter test

# View solutions
cd <category>/solutions/
```
