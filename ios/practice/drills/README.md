## Extensions Drills

Practice writing Swift extensions for common tasks.

### Example: String Extensions

```swift

// Add these methods to String via extension:
// 1. isPalindrome -> Bool
// 2. toTitleCase() -> String
// 3. initials() -> String
// 4. isValidEmail() -> Bool
```

## How to Use

1. Pick a category based on what you want to practice
2. Set a timer for the suggested time
3. Complete the exercise
4. Compare with reference solution
5. Move to next drill
